      Javascript 이론 중 깊이있게 탐구한 주제

      Javascript 비동기 패턴에 쓰이는 Promise, async/await 개념을 이해하기 위해 직접 이들을 구현해 보았습니다.
      JS의 prototype과 class 관계 이해를 위해 prototype을 이용하여 클래스 상속을 구현해 보았습니다.
      JS의 이벤트 위임이 필요한 이유를 파악하기 위해 이벤트 객체의 내부 동작 방식에 대해 분석해 보았습니다.

      VanillaJS로 React 환경 모방

      React 컴포넌트 이해를 위해 VanillaJS로 컴포넌트를 직접 구현해 보았습니다.
      React의 상태관리 라이브러리인 Redux를 직접 구현해 보았습니다.
      React의 SPA 라우터인 react-router를 직접 구현해 보았습니다.
      React의 비동기 통신 라이브러리인 axios를 직접 구현해 보았습니다.
